📖 Transformation Tricks

Quick reference for solving Translations, Reflections, and Rotations · Pre-Algebra Ch. 3.1 – 3.3

➤ TRANSLATIONS (Slide)

The shortcut: Just add or subtract the same numbers from every vertex.

  • Right = add to x · Left = subtract from x
  • Up = add to y · Down = subtract from y
  • Rule form: (x, y) → (x + a, y + b), where positive values mean right/up and negative values mean left/down.
  • To find the rule when given before and after: subtract the originals from the images. If A(2, −1) → A′(−1, −5), then x changed by −3 and y changed by −4, so the rule is (x, y) → (x − 3, y − 4).
  • Sanity check: every vertex must move by the same amount in the same direction. If they don't, it isn't a translation.
➤ REFLECTIONS (Flip)

The shortcut: Flip the sign of whichever coordinate the line of reflection controls.

  • Across the x-axis → flip the y: (x, y) → (x, −y)
  • Across the y-axis → flip the x: (x, y) → (−x, y)
  • Across a horizontal line y = k → (x, y) → (x, 2k − y). Easier: count how far above or below the line each point is, then put the image the same distance on the other side.
  • Across a vertical line x = k → (x, y) → (2k − x, y). Same logic — count distance to the line, mirror it.
  • To find the line of reflection: take a point and its image, find the midpoint of the segment connecting them. The line of reflection passes through that midpoint perpendicular to that segment.
  • Sanity check: the image is the same size and shape, but "backwards" — orientation reverses (like a mirror image).
➤ ROTATIONS (Turn — about the origin in this chapter)

The shortcut: Memorize four rules. All counterclockwise rotations about the origin.

RotationRuleWhat it does
90° CCW(x, y) → (−y,  x)Swap, then negate the new x
180°(x, y) → (−x, −y)Negate both coordinates
270° CCW(x, y) → ( y, −x)Swap, then negate the new y
360°(x, y) → ( x,  y)Back to where you started

Clockwise conversion trick: A clockwise rotation of n° = a counterclockwise rotation of (360 − n)°.

  • 90° CW = 270° CCW → use (y, −x)
  • 270° CW = 90° CCW → use (−y, x)
  • 180° is the same either way — direction doesn't matter for a half-turn.

Mental shortcut for 90°: swap the coordinates, then flip the sign of one of them. For 90° CCW it's the new x that gets the negative; for 90° CW it's the new y.

To identify an unknown rotation: plug one vertex into each of the four rules and see which produces the image point. If A(1, 2) → A′(−2, 1), test (−y, x) = (−2, 1) ✓ — that's 90° CCW.

➤ UNIVERSAL TEST-DAY TIPS
  1. Work one vertex at a time. Apply the rule, write the new point, move on. Don't try to do all of them in your head at once.
  2. Count grid lines as a check. After computing, look at the picture and verify the image landed where it should.
  3. Read the direction word carefully. "Left/right/up/down" for translations, "in/across the line" for reflections, "clockwise/counterclockwise" for rotations. Mixing these up is the #1 mistake.
  4. Use prime notation (A′, B′, C′) for image points — the problem expects it, and it keeps original and image clearly separate.
  5. Original and image are always identical in size and shape. Only position or orientation changes. If your image came out a different size, something went wrong.
